# sjce-leave-system

## Install requirements
```
pip install -r requirements.txt
```

## Run app
```
python app.py
```

### Leave type encoding
```
ll - leave-with-leave letter
ml - Medical leave
pl - Priviledge leave
ab - Absent
```

### Admin encoding
```
1 - Counselor
2 - HOD
3 - Year In-charge
```
